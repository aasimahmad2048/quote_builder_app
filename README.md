 
 
# Quote Builder App

This is a Flutter application designed to help users build and manage quotes.

## Features

- **Create Quotes:** Easily generate new quotes with various items and details.
- **Manage Items:** Add, edit, and delete items within a quote.
- **Calculate Totals:** Automatically calculates the total cost of a quote.
- **Save and Load Quotes:** Persist your quotes for future access.
